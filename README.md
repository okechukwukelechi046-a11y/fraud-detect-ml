# fraud-detect-ml
Production-ready fraud detection system with real-time ML inference and model management.

A production-ready machine learning system for real-time fraud detection in financial transactions. Features ensemble modeling, online feature serving, and comprehensive monitoring.

Features
Real-time Inference - <50ms prediction latency at scale

Model Ensemble - XGBoost, LightGBM, and Neural Network with weighted voting

Feature Store - Feast-powered online feature serving

Concept Drift Detection - Automated detection of changing patterns

Explainable AI - Feature importance and risk factor explanations

A/B Testing - Framework for model experimentation

Monitoring Dashboard - Real-time metrics and performance tracking

Feedback Loop - Continuous learning from labeled fraud cases

Tech Stack
Python 3.9+ - Core programming language

Scikit-learn, XGBoost, LightGBM - Traditional ML models

PyTorch - Deep learning framework

FastAPI - High-performance model serving

Feast - Feature store for consistency

MLflow - Experiment tracking and model registry

Redis - Low-latency caching

Docker - Containerization

Prometheus/Grafana - Monitoring and visualization
